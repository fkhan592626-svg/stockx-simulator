# StockX Simulator 📈

A production-grade Real-Time Stock Trading and Portfolio Analytics Platform built for software engineering interviews.

## 🚀 Tech Stack

### Backend
- Node.js + Express.js + TypeScript
- PostgreSQL (Neon) + Prisma ORM
- JWT Authentication + Refresh Tokens
- Socket.io for Real-Time Updates
- Finnhub WebSocket for Live Stock Prices

### Frontend
- React.js + TypeScript
- Tailwind CSS
- Recharts for Analytics
- Socket.io Client

## ✨ Features

- 🔐 JWT Authentication with Refresh Tokens
- 📊 Real-Time Stock Prices via Finnhub WebSocket
- 💹 Order Matching Engine (Market & Limit Orders)
- 📈 Portfolio Analytics Dashboard
- 👀 Watchlist Management
- 📋 Transaction History
- 🔌 Socket.io Live Updates
